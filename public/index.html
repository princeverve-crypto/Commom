Core Capabilities

📡

Real-Time GPS Tracking

Live vehicle positioning via Teltonika FMC130/FMC640 devices with 10-60s update intervals. Geofence alerts and route optimization included.

⛽

Fuel Theft Detection

AI-powered anomaly detection flags sudden fuel drops. Multi-factor analysis includes volume variance, time windows, and location correlation.

👤

Driver Monitoring

Behavioral scoring system tracks harsh braking, speeding, excessive idling. Weekly improvement trends and driver performance rankings.

🚨

Real-Time Alerts

Instant SMS/Push notifications for critical events. Alert deduplication prevents spam while maintaining sensitivity to genuine threats.

📊

Advanced Analytics

Comprehensive dashboards with 7-day history, KPI tracking, and predictive maintenance alerts based on vehicle performance patterns.

⚙️

Seamless Integration

Native Wialon API integration handles device ingestion, CAN parsing, and rule engine. Custom backend focuses on business logic.

System Architecture

JOSYEL leverages Wialon's enterprise telematics infrastructure while building a custom UI and intelligence layer focused on fuel theft detection and operational efficiency.

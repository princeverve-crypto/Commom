# Caveman Skills Setup

This branch documents the setup of the caveman plugin for Claude Code.

## Steps Completed

1. Created directory: `~/.claude/skills/caveman`
2. Added caveman marketplace: `JuliusBrussee/caveman`
3. Installed caveman plugin: `caveman@caveman`
4. Cloned repository from `https://github.com/JuliusBrussee/caveman.git`
5. Copied skill files to `~/.claude/skills/caveman/`

## Usage

The caveman skill is now fully set up and available in your Claude Code environment. Use `/caveman` to activate caveman mode for ultra-compressed communication.
